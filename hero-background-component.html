Usage Instructions

Key Classes:

  • .bg-aurora-hero - Aurora gradient background
  • .has-grain - Base grain effect
  • .grain-hero - Enhanced grain for hero sections
  • .hero-card - Main hero container
  • .hero-card__content - Text content area
  • .hero-card__title - Hero headline styling

Customization:

  • Modify CSS variables at the top for spacing and colors
  • Adjust gradient colors in .bg-aurora-hero
  • Change grain intensity by modifying opacity and filter values
  • Update responsive breakpoints as needed
